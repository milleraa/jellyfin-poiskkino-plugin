# Features

Каждая фича — папка `docs/backlog/features/<feature-name>/`.

## Шаблон

Скопируйте [feature-template](feature-template/) в новую папку с именем в **kebab-case**.

## Правила имён

- Только латиница, цифры, дефис: `user-auth-oauth2`.
- Коротко и уникально в рамках репозитория.
