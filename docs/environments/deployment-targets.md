# Deployment targets

*Источники: [`README.md`](../../README.md), подтверждение Стейхолдера (2026-05-18).*

Продукт — DLL-плагин для Jellyfin. Отдельных stage/prod-серверов для разработки нет.

| Окружение | URL / путь | Кто деплоит | Примечание |
|-----------|------------|-------------|------------|
| **dev** | Локальный Jellyfin: каталог плагинов (Linux `/var/lib/jellyfin/plugins/`, Docker `/config/plugins/`, и т.д.) | Стейхолдер | Сборка локально или из артефакта CI после настройки Actions |
| **stage** | N/A | — | Не используется |
| **prod** | Jellyfin пользователя (тот же механизм — копирование DLL) | Администратор | Ручная установка; без централизованного deploy из репо |

**Итог:** разработка и проверка — только на локальном Jellyfin; общего dev/stage-стенда нет.
