# Deployment targets

*Источники: [`README.md`](../../README.md) (установка плагина).*

Отдельных dev/stage/prod URL для приложения нет: продукт — DLL-плагин, устанавливаемый в экземпляр Jellyfin администратором.

| Окружение | URL / путь | Кто деплоит | Примечание |
|-----------|------------|-------------|------------|
| **local / dev** | Каталог плагинов Jellyfin (напр. Linux: `/var/lib/jellyfin/plugins/`, Docker: `/config/plugins/`) | Разработчик / админ | `dotnet build -c Release`, копирование DLL, перезапуск Jellyfin |
| **stage** | N/A | — | Не выделено в репозитории |
| **prod** | Экземпляр Jellyfin пользователя | Администратор Jellyfin | Не централизованный деплой из этого репо |

**Итог:** окружений как сервисов нет — только локальная/ручная установка в Jellyfin.
