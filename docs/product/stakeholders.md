# Stakeholders

*Источники: подтверждение Стейхолдера (2026-05-18), `git remote` → `milleraa/jellyfin-poiskkino-plugin`.*

| Роль | Имя / команда | Зона ответственности |
|------|----------------|----------------------|
| Стейхолдер (владелец продукта) | Владелец репозитория (`milleraa`) | Приоритеты, приёмка PR/MR, API-ключи ПоискКино |
| Maintainer / разработчик | Тот же (единственный участник) | Код плагина, сборка и установка DLL, настройка CI |
| Пользователи | Администраторы Jellyfin | Установка плагина, настройка API-ключа в Dashboard |

**Приёмка:** все решения по продукту, merge и релизам — у Стейхолдера.
