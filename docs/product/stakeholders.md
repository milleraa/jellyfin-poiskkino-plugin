# Stakeholders

*Источники: шаблон agile-процесса, [`git remote`](../../README.md) (владелец репозитория на GitHub — уточнить у Стейхолдера).*

| Роль | Имя / команда | Зона ответственности |
|------|----------------|----------------------|
| Стейхолдер (владелец продукта) | *уточнить* | Приоритеты, приёмка PR/MR, API-ключи ПоискКино для prod |
| Maintainer / разработчик | *уточнить* (репозиторий: `milleraa/jellyfin-poiskkino-plugin`) | Код плагина, релизы DLL |
| Пользователи | Администраторы Jellyfin | Установка плагина, настройка API-ключа в Dashboard |
